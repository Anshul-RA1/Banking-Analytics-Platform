# Banking Analytics Platform

## Overview

The Banking Analytics Platform is a production-style end-to-end data analytics project designed to simulate a real banking implementation using SAS, PostgreSQL, and Power BI.

The project follows the complete Software Development Life Cycle (SDLC) including:

* Requirement Gathering
* Business Analysis
* Data Modeling
* SQL Development
* SAS ETL Development
* Testing
* UAT
* Deployment
* Documentation
* Dashboard Development

---

## Business Problem

Banking stakeholders often struggle to obtain a consolidated view of customer profitability, loan performance, branch performance, and customer risk metrics.

This project aims to provide a centralized analytics solution that enables management to make data-driven decisions.
